# Installation
**Recommended**: Download From [Releases](https://github.com/RustyBalboadev/goGrab/releases/tag/1.0)

Installing From Source
```
git clone https://github.com/rustybalboadev/goGrab
cd goGrab
go build -ldflags -H=windowsgui
```
